## 🎯 使用场景示例

### 1️⃣ 最简单的用法（仅视频）

```markdown
{% dplayer "url=/videos/baby-video.mp4" %}
```

**说明**：
- 只需要视频路径
- 播放前会显示黑屏或视频第一帧
- 适合快速添加视频

---

### 2️⃣ 推荐用法（视频 + 封面图）

```markdown
{% dplayer "url=/videos/baby-laugh.mp4" "pic=/img/baby-laugh-cover.jpg" %}
```

**说明**：
- `url`: 视频文件路径
- `pic`: 封面图路径（播放前显示）
- 封面图让视频更美观，避免黑屏
- **这是最推荐的用法**

---

### 3️⃣ 完整参数用法

```markdown
{% dplayer "url=/videos/first-step.mp4" "pic=/img/first-step-cover.jpg" "autoplay=false" "loop=false" "volume=0.7" %}
```

**参数说明**：
- `url`: 视频路径（必需）
- `pic`: 封面图路径（推荐）
- `autoplay`: 自动播放（true/false，默认 false）
- `loop`: 循环播放（true/false，默认 false）
- `volume`: 音量（0-1，默认 0.7）

## 📂 文件路径说明

### 视频文件
- **上传到**：`/var/www/hexo-blog/source/videos/`
- **引用路径**：`/videos/文件名.mp4`

### 封面图片
- **上传到**：`/var/www/hexo-blog/source/img/`
- **引用路径**：`/img/文件名.jpg`

**示例**：
```bash
# 服务器上的实际位置
/var/www/hexo-blog/source/videos/2025-04-20-rollover.mp4
/var/www/hexo-blog/source/img/2025-04-20-rollover-cover.jpg

# 在文章中的引用路径
url=/videos/2025-04-20-rollover.mp4
pic=/img/2025-04-20-rollover-cover.jpg
```

## 🎨 封面图制作建议

### 封面图来源

1. **从视频中截图**（推荐）
   - 选择视频中最精彩的一帧
   - 使用视频编辑软件截图
   - 保持视频的比例（推荐 16:9）

2. **使用专门拍摄的照片**
   - 与视频内容相关的照片
   - 更清晰、更美观

### 推荐尺寸

- **16:9 比例**：1920x1080、1280x720
- **4:3 比例**：1600x1200、1024x768

### 文件格式

- 推荐：**JPG** (文件小，加载快)
- 也可用：PNG (质量更好，但文件大)

## ⚠️ 注意事项

### 1. 视频格式

- **推荐格式**：MP4 (兼容性最好)
- **编码**：H.264
- **其他支持**：WebM、OGG

### 2. 视频大小

- 建议单个视频 < 100MB
- 如果视频过大，建议压缩后上传
- 可使用在线工具或软件压缩

### 3. 路径规则

- 视频路径**必须**以 `/videos/` 开头
- 封面图路径**必须**以 `/img/` 开头
- 文件名**区分大小写**

### 4. 上传步骤

```bash
# 1. 上传视频到服务器
scp 2025-04-20-rollover.mp4 root@服务器IP:/var/www/hexo-blog/source/videos/

# 2. 上传封面图到服务器
scp 2025-04-20-rollover-cover.jpg root@服务器IP:/var/www/hexo-blog/source/img/

# 3. 在文章中引用
# url=/videos/2025-04-20-rollover.mp4
# pic=/img/2025-04-20-rollover-cover.jpg

# 4. 重新生成网站
npx hexo clean && npx hexo generate
```

## 🔧 常见问题

### Q: 视频无法播放？

**检查清单**：
1. ✅ 视频是否上传到 `source/videos/` 目录
2. ✅ 路径是否以 `/videos/` 开头
3. ✅ 文件名是否正确（区分大小写）
4. ✅ 视频格式是否为 MP4
5. ✅ 是否重新生成了网站

### Q: 封面图不显示？

**检查清单**：
1. ✅ 封面图是否上传到 `source/img/` 目录
2. ✅ 路径是否以 `/img/` 开头
3. ✅ 文件名是否正确
4. ✅ 图片格式是否为 JPG/PNG

### Q: 视频加载很慢？

**解决方案**：
1. 压缩视频文件大小
2. 降低视频分辨率
3. 调整视频比特率

### Q: 想要视频自动播放？

```markdown
{% dplayer "url=/videos/video.mp4" "autoplay=true" %}
```

**注意**：大多数浏览器会阻止自动播放有声音的视频，建议设置为 `autoplay=false`

## 📊 参数速查表

| 参数 | 说明 | 必需 | 默认值 | 示例 |
|------|------|------|--------|------|
| `url` | 视频路径 | ✅ | 无 | `url=/videos/video.mp4` |
| `pic` | 封面图路径 | ❌ | 无 | `pic=/img/cover.jpg` |
| `autoplay` | 自动播放 | ❌ | false | `autoplay=true` |
| `loop` | 循环播放 | ❌ | false | `loop=true` |
| `volume` | 音量（0-1） | ❌ | 0.7 | `volume=0.5` |

## 💝 小贴士

1. **封面图很重要**：好的封面图能吸引点击，建议每个视频都添加封面
2. **文件命名规范**：使用日期前缀，方便管理和查找
3. **视频时长控制**：建议单个视频不超过 5 分钟
4. **批量上传**：多个视频可以使用 FTP 工具批量上传
5. **备份重要视频**：建议在本地也保留一份视频备份
